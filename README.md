# scraptop
Scraping tokopedia products using scrapy

#HOW TO INSTALL
setup your virtualenv

```
git clone https://github.com/xhijack/scraptop
pip install -r requirement.txt
```

#HOW TO RUN
```
cd scraptop
```

```
scrapy crawl tokopedia -a keyword=iphone
```
